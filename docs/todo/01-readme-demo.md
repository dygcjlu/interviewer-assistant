# #1 README + Demo

## 目标

让开发者和招聘者在 GitHub 上扫 2 分钟内，能看懂项目是什么、能做什么。

## 范围

- 编写完整 README，包含：项目简介、架构图（复用 `docs/arc/overview.md` 的 mermaid）、功能截图或 GIF、一键启动说明
- 录制一个 GIF，涵盖完整主流程：上传简历 → 生成面试题 → 面试转写 → 生成评价报告
- 平台说明：当前仅支持 Windows（WASAPI），非 Windows 用户标注"暂不支持"；跨平台支持作为后续独立升级点

## 验收条件

- [ ] README 有项目简介（一句话定位 + 功能列表）
- [ ] README 嵌入架构图（mermaid 或导出图片）
- [ ] README 嵌入一个 GIF，完整展示：上传简历 → 生成面试题 → 面试实时转写 → 追问建议弹出 → 生成评价报告
- [ ] README 有"快速开始"章节：环境要求、依赖安装、启动命令
- [ ] README 明确标注当前仅支持 Windows，非 Windows 暂不支持
- [ ] GIF 文件大小合理（建议压缩到 10MB 以内），能在 GitHub 页面正常加载
